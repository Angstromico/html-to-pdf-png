HTML to PDF & PNG Converter

Convert any HTML file to PDF or high-quality PNG images

Quick Start

Run these commands to convert this example:

npm run example:pdf  # Convert to PDF
npm run example:png  # Convert to PNG
npm run example:all  # Convert to both

Features

PDF Output

A4 format with headers, footers, and page numbers

PNG Output

Full page screenshot at 1920x1080 resolution

Mermaid Support

Renders diagrams as vector graphics

CLI Interface

Specify input file, output directory, and filename
